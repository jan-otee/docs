---
title: "Install a Runtime"
description: "Deploy and configure your first service."
---

After following the installation instructions successfully in your device's terminal, switch back to the web app. You should now see a prompt on your device page to install your first runtime (try refreshing the page if you don't).

<img
  className="block dark:hidden mx-auto"

  width="400"
/>

<img
  className="hidden dark:block mx-auto"

  alt="Runtime 1 Dark"
  width="400"
/>

Follow the call to action and type a runtime name of your choice in the 'Service ID' input field - leave all other settings as is for now and hit 'Deploy'. This will prompt your device agent to download all necessary files to set up the runtime. Congratulations, your first virtual PLC is now live!
